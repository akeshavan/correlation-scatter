MRIQC & Mindcontrol Data Exploration

This dataviz is modified from GitHub user d-miller's excellent example.

The data comes from the Healthy Brain Network dataset, where MRIQC was run on the T1w images by Jamie Hanson .

The last row in the scatter plot below (called "mean") is the average quality control score from crowdsourced QC of the HBN data on Mindcontrol

A mean score of -5 means that the user confidently believes the image fails QC.

A mean score of +5 means the user (or users) confidently believe the image passes QC.
